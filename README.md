# Proyecto React (Vite)

Estructura base del Proyecto de Servicio Comuitario (Sistemas de Informacion).

## Estructura

```
web-react/
  package.json
  vite.config.js
  index.html
  README.md
  .gitignore
  .eslintrc.cjs
  src/
    main.jsx
    App.jsx
    components/
    pages/
    assets/
    styles/
      global.css
```

## Scripts

- `npm run dev`: Inicia servidor de desarrollo
- `npm run build`: Construye produccion en `dist/`
- `npm run preview`: Previsualiza la compilacion
